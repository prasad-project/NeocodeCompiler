## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/NeoCompiler.git
cd NeoCompiler
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [https://neocode-compiler.vercel.app/] in your browser.
